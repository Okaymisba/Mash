# Mash Syntax Guide
## 1. Variables
Mash allows variable assignment using =.
```
x = 10;
y = 5;
```

## 2. Arithmetic Expressions
Mash supports basic arithmetic operations:
```
a = 10 + 5;      # Addition
b = 10 - 3;      # Subtraction
c = 10 * 2;      # Multiplication
d = 10 / 2;      # Division
```

## 3. Printing Output
Use print to display values:
```
print(x)         # Outputs: 10
print(a + b)     # Outputs: result of 15 + 7
```
